Вступ 2026 (Кафедра АВКТ)
Веб-сторінка (інформаційний листок), розроблена для абітурієнтів спеціальності «Автоматизація, комп’ютерно-інтегровані технології та робототехніка». 
Проєкт створений для популяризації освітніх програм та інформування про вступ до бакалаврату.

📋 Огляд проєкту
Цей лендінг є цифровим інформаційним буклетом, який містить ключову інформацію про вебінари кафедри АВКТ, освітні рівні та деталізований опис напрямків підготовки.

Основні розділи:
Інтерактивний вибір напрямків: реалізовано за допомогою стилізованих чекбоксів з динамічною зміною стану тексту.
Інформаційний блок вебінару: чіткий розклад (дата, час) та пряме посилання на платформу Zoom.
Контакти та доступ: швидкий доступ до вебінару через ідентифікатор, пароль або QR-код.

## 🛠 Технологічний стек

Проєкт реалізований з використанням сучасних методів верстки:
HTML5: семантична структура документа.
CSS3 (Custom Properties): використання змінних (`:root`) для легкого керування кольоровою палітрою (Brand Colors: `#1A3A8F`, `#C8A84B`).
Flexbox: для створення адаптивної та гнучкої сітки (картки спеціальностей, вирівнювання контенту).
SVG & Graphics: використання оптимізованих зображень та іконок для високої швидкості завантаження.
